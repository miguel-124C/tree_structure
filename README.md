

# Steps to running project localhost

0. Clone to project
```https://github.com/miguel-124C/tree_structure```

1. Created environment virtual
``` python -m venv venv ```

2. Activated

- 1. In Windows
``` .\venv\Scripts\Activate ```
- 2. In Linux/Mac
``` source venv/bin/activate ```

3. Execute app
``` py app.py ```
